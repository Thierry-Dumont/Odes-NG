Vector class (of doubles). 

Vector class (of doubles) O\-F F\-I\-X\-E\-D S\-I\-Z\-E n. We want to use fortran indexing (from 1 to...)\-: this is just a wrapper around an array of doubles. Note\-: this is not a very safe class!.
